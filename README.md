# Expense Tracker

A responsive web application built with **HTML**, **CSS**, and **JavaScript** to help users manage their personal finances by tracking income and expenses.

## Planned Features

- Add income and expense transactions.
- Edit and delete transactions.
- Calculate total balance.
- Display total income and total expenses.
- Categorize transactions.
- Store data using Local Storage.
- Responsive design.
- Form validation.
- Transaction history.
